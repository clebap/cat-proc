
**Código de clasificación:** C061

**Familia:** PROTECCIÓN DE DATOS

### OBJETO / DESCRIPCIÓN

Permite a los ciudadanos contactar directamente con el Delegado de Protección de Datos del Ayuntamiento para resolver dudas, consultas o incidencias relacionadas con el tratamiento de datos personales por parte de la administración municipal.
